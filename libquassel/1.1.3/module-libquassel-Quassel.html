libquassel~ Quassel

new Quassel(server, port, optionsopt, loginCallback)

Main class to interact with Quassel instance. It extends EventEmitter2

Parameters:
Name Type Attributes Description
server string

The server hostname or IP address
port number

The port on which runs Quassel on the server
options Object <optional>

Allows optionnal parameters

Properties
Name Type Attributes Default Description
nobacklogs boolean <optional>
 false

Do not request backlogs (mostly for debug purpose)
initialbackloglimit number <optional>
 options.backloglimit

number of backlogs to request per buffer at connection
backloglimit number <optional>
 100

number of backlogs to request per buffer after connection
securecore boolean <optional>
 true

Use SSL to connect to the core (if the core allows it)
highlightmode module:libquassel~Quassel.HighlightModes <optional>
 0x02

Choose how highlights on nicks works. Defaults to only highlight a message if current nick is present.
loginCallback module:libquassel~Quassel~loginCallback
Source:
  • libquassel.js, line 59
Example
var quassel = new Quassel("localhost", 4242, {}, function(next) {
  next("user", "password");
});
quassel.connect();
