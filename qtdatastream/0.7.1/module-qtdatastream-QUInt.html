QUInt

qtdatastream~ QUInt

qtdatastream.js:391

new QUInt(obj)

This class allow users to force a specific object to be recognized as a QUInt for the Writer. Javascript number are converted to QUInt objects internally. When parsed from reader, QUInt objects are converted back to Javascript number
